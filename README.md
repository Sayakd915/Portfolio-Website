# Full Stack Machine Learning Developer Portfolio

A modern, responsive portfolio website built with React and TailwindCSS.

## Features

- **Navbar**: Fixed navigation with smooth scrolling
- **Banner**: Eye-catching hero section with gradient background
- **About**: Personal introduction section
- **Education**: Academic background with timeline
- **Skills**: Interactive skill bars organized by category
- **Experience**: Professional experience timeline
- **Projects**: Showcase of ML and full-stack projects
- **Footer**: Contact information and social media links

## Technologies Used

- React 18
- TailwindCSS 3
- React Scripts

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
